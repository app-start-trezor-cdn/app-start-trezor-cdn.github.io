Trezor.io/Start® | Starting Up Your Device | Trezor®

To update your Trezor Suite to version 22.9.3 and ensure you have the latest features and improvements, follow these steps:

1. Check for Update Notification

If you have Trezor Suite open, it should notify you of the update automatically. If you see a notification about the update to version 22.9.3, you can proceed directly to download and install it.

2. Download and Install the Update

  • Open Your Browser: Go to the official Trezor website by visiting trezor.io/start.
  • Download the Update: Follow the instructions to download the latest version of Trezor Suite. If the update is not available directly from your current installation, you might need to download the installer from the website.
  • Install the Update: Run the installer and follow the on-screen prompts to update Trezor Suite to version 22.9.3.

3. Install Trezor Bridge (if necessary)

If you haven’t installed Trezor Bridge yet:

  • Download Trezor Bridge: Follow the instructions on trezor.io/start to download and install Trezor Bridge. This software is necessary for your Trezor device to communicate with Trezor Suite.
  • Install the Software: Run the downloaded installer and follow the installation prompts.

4. Restart Trezor Suite

After installation, restart Trezor Suite to ensure all updates are properly applied.

5. Explore New Features

With the new version, you’ll benefit from:

  • Enhanced Security: Improved measures to protect your assets.
  • New Design: A refreshed user interface for better navigation.
  • Improved Privacy: New features to help manage your privacy and control your UTXOs (Unspent Transaction Outputs).

6. Configure Your Device

Once the update is complete:

  • Connect Your Trezor Device: Plug your Trezor device into your computer.
  • Open Trezor Suite: Launch Trezor Suite if it’s not already open.
  • Follow Instructions: The software will guide you through any additional setup or configuration required.

7. Explore Features

  • Desktop or Browser Access: Trezor Suite can be used on both desktop and browser. Visit suite.trezor.io to access the Suite in your browser.
  • Discover New Features: Explore the new features and improvements introduced in version 22.9.3.

By following these steps, you will ensure your Trezor Suite is up-to-date and you can take advantage of the latest security enhancements and features.
